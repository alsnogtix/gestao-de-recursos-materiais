Instruções para uso parcial do site

A implementação do site se encontra em fase parcial.
Os dados estão sendo gravados provisoriamente em localStorage no navegador, o navegador usado para testes até a etapa atual é o firefox.
Como os dados estão sendo gravados de maneira provisoria a tabela que mostra os dados salvos apresentam dados do tipo 'undefined', isso ocorre quando, por exemplo, gravamos um item do tipo lápis e clicamos na aba caderno na tela de material escolar.
Outro detalhe a destacar é o fato do id não estar sendo mostrado corretamente, para mostrar o numero correto de dados guardados foi criado um id provisorio em localStorage.
Para acessar os dados em localStorage no FireFox pressione 'F12' -> aba storage -> localStorage.
