Descrição do sistema

Os dados estão sendo gravados em sessionStorage no navegador, o navegador usado para testes até a etapa atual é o firefox.
O sistema conta com 5 páginas(Inicio, Material de Limpeza, Material Escolar, Material Didatico e Infomrações).
A pagina "Inicio" mostra um relatório com todos os itens cadastrados no sistema, agrupados por tipo. É possível selecionar o subtipo e realizar uma busca.

As páginas "Material de Limpeza", "Material Escolar" e "Material Didatico" mostram os formulários que possibilitam o gerenciamento de itens no sistema. cada pagina
contém as abas que correspondem aos subtipos solicitados na definição do trabalho, o formulário para inserir os dados e uma tabela mostrando os dados cadastrados.
Para cada página temos um botão que limpa os campos preenchidos, os campos do formulário e dois botões (salvar e buscar). O botão "salvar" grava um novo dado em
sessionStorage caso não exista o dado informado, caso o dado já exista no banco o sistema atualiza o dado mantendo o id. O botão "buscar" verifica
se existe um dado gravado com as informações contidas no formulário e mostra na tabela abaixo, caso não existir, é mostrado todos os dados.
A Tabela mostrada abaixo do formulário mostra todos os dados cadastrados. clicando na tabela é possível editar os dados do registro clicado
Para apagar um dado existe um botão com icone de lixeira para cada item na tabela mostrada, ao clicar no botão e confirmando a exclusão
o dado será apagado de sessionStorage.

A página "informações" mostra instruções de uso do sistema e links para acessar dados relevantes ao sistema.
